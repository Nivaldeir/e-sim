# Templates de Email

Este diretório contém os templates EJS para envio de emails do sistema.

## Template: Document Expiration

Template para alertas de vencimento de documentos.

### Variáveis disponíveis:

- `templateName`: Nome do template do documento
- `organizationName`: Nome do órgão
- `companyName`: Nome da empresa
- `establishmentName`: Nome do estabelecimento
- `responsibleName`: Nome do responsável
- `expirationDate`: Data de vencimento formatada
- `alertDate`: Data de alerta formatada (opcional)
- `status`: Status do documento
- `customData`: Dados customizados do documento (objeto)
- `observations`: Observações do documento
- `systemUrl`: URL base do sistema
- `documentId`: ID do documento
